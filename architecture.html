NFC / NTAG213

觸發介面，每張貼紙燒入唯一 SN 的完整 URL

URL SPEC
NFC / QR Code URL 規格
每張 NFC 貼紙燒入以下格式的 URL，SN 對應試算表中的設備序號。
URL 格式
https://script.google.com/macros/s/AKfycby.../exec?sn={SN}
完整範例
https://script.google.com/macros/s/AKfycbyjrhodTovCfPsBRnm2vgPvIMconAToVxxO5IEy4T24PVSSdJkg9zggUfLjzenJqv-_/exec?sn=01000011001
參數 說明 範例
sn 設備唯一序號，對應試算表 SN 欄 01000011001
NFC 貼紙規格：NTAG213（144 bytes）．一張貼紙 = 一個 SN．URL 不變 = NFC 不需重燒
